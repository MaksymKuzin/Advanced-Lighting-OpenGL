# OpenGL 3D Model Viewer with Blinn-Phong Lighting

## Опис проєкту

Даний програмний засіб розроблений мовою C++ із використанням бібліотеки OpenGL та призначений для візуалізації тривимірних моделей у форматі GLB із застосуванням моделі освітлення Блінна–Фонга (Blinn-Phong Lighting).

Програма забезпечує інтерактивний перегляд 3D-об'єктів, масштабування, обертання, переміщення сцени та перемикання між декількома моделями.

Проєкт був створений у межах виконання дипломної роботи, присвяченої дослідженню методів освітлення та відображення тривимірної графіки в реальному часі.

---

## Основні можливості

* Завантаження 3D-моделей формату GLB.
* Відображення моделей за допомогою OpenGL.
* Реалізація моделі освітлення Blinn-Phong.
* Підтримка текстур та матеріалів.
* Масштабування сцени колесом миші.
* Обертання моделі за допомогою миші.
* Переміщення моделі середньою кнопкою миші.
* Перемикання між моделями.
* Відображення статистики моделі (кількість вершин та полігонів).

## Алгоритм роботи

1. Ініціалізація GLFW та OpenGL.
2. Завантаження шейдерів.
3. Завантаження 3D-моделей через Assimp.
4. Створення буферів вершин та індексів.
5. Розрахунок освітлення за моделлю Блінна–Фонга.
6. Відображення сцени.
7. Обробка взаємодії користувача.

## Результат

Програмний засіб демонструє можливості сучасного графічного конвеєра OpenGL для візуалізації тривимірних моделей та дослідження методів освітлення реального часу.
